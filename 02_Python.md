## 주요 시각화 사이트

| 사이트명           | 특징 및 설명                                                                                  |
|-------------------|----------------------------------------------------------------------------------------------|
| **TIOBE Index**   | 월별로 전 세계 프로그래밍 언어 인기 순위와 점유율을 표와 그래프로 시각화.                      |
| **Visual Capitalist (Voronoi App)** | GitHub 데이터를 기반으로 연도별 인기 언어 순위를 인터랙티브 차트로 시각화.            |
| **GitHub Octoverse** | GitHub 내 언어별 활동량, 성장률, AI 프로젝트 기여 등을 다양한 그래프로 제공.                   |
| **RedMonk**       | 개발자 커뮤니티와 GitHub 데이터를 결합해 언어 순위를 플롯 차트로 시각화.                        |
| **Statista**      | 글로벌 설문조사 기반 언어 사용률을 막대그래프 등으로 시각화.                                    |

### 참고할 만한 시각화 링크
- TIOBE Index: 월별 인기 언어 순위 그래프
- Visual Capitalist: 연도별 GitHub 인기 언어 차트
- GitHub Octoverse: 언어별 성장률·기여도 대시보드
- RedMonk: 개발자 활동 기반 언어 순위 플롯

이들 사이트에서 최신 순위와 트렌드를 직관적으로 확인할 수 있습니다.
